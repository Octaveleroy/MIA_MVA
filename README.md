# MIA_MVA
Projet du MVA pour Medical Image Analysis. Implémentation de [Label Refinement Network from Synthetic Error Augmentation for Medical Image Segmentation](https://arxiv.org/pdf/2209.06353)

# DataSet
Il faut télécharger le [dataset](https://www.kaggle.com/datasets/andrewmvd/drive-digital-retinal-images-for-vessel-extraction) et le mettre dans le même dossier que le notebook (DRIVE au même niveau que le notebook).
